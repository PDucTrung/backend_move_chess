# Move Chess Backend
## Run the script locally
- Make sure you have MongoDB and Node (v18.18.2 or later) installed
- Run **yarn install** to install all required libraries
- Create .env file in the root folder
```
NODE_ENV=
PORT=
API_BASE_URL=
SESSION_SECRET=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

DB_CONNECTOR=
MONGO_HOST=
MONGO_PORT=
MONGO_DB_NAME=
MONGO_USER=
MONGO_PASSWORD=

ADMIN_EMAIL=
ADMIN_EMAIL_PASS=

JWT_ACCESS_SECRET=
JWT_REFRESH_SECRET=
EMAIL_SECRET=

OPENAI_API_KEY=
```
